3.3 Map 接口详解

Map接口是 Java 集合框架中用于存储键值对（Key - Value）数据的接口，它提供了一种通过键来快速访问对应值的数据结构，就像我们日常生活中的字典，通过单词（键）可以查到对应的释义（值）。
